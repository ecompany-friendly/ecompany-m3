## Funcionalidades da pagina
  - Página Login;
  - Página Cadastro; 
  - Página Dashboard;  
  - Intregração feita por api;
  - Rotas com permissões de recurso;
  - Usuário pode buscar, cadastrar, editar e deletar;

## Instalação

    - Repositorio
    $ git clone https://github.com/jveiiga/ecompany-friendly.git

    - Dependencias
    $ yarn add 
    @ytypescript -D
    @styled-components
    @react-icons
    @lord-icon-element
    @react-modal
    @react-hook-form
    @yup
    @react-router-dom

    - Rodar aplicativo
    $ yarn start
